# MERN

```
npm init "asdf"
```

### nodemod - monitor changes like sass

```
npm install nodemod
```

add this to packages.json

```javascript
  "scripts": {
    "start": "nodemon app.js",
```

### Run server

```
npm start
```

```
npm install express
```
